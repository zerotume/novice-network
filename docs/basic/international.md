# 国际服

《最终幻想14》在运营上大致可分为**国服**、**国际服**、**韩服**和**繁中服**四方。这四者由四个不同的公司运营：国服由盛大（盛趣）运营，国际服由游戏开发商 SquareEnix（SE） 自己运营，韩服由被盛大收购的韩国老牌游戏运营商 Actoz Soft 运营，繁中服由宇峻奧汀代理运营。

## 国服和其他服的区别

国服和其他服的游戏内容是完全一致的，入坑后您均可参照本站的其他相关信息进行游戏。

|  | 国服 | 其他服 |
|--|-------|-----|
| 更新时间 | 标准 | 国际服同步；韩服滞后1个小版本，预计即将同步；繁中服滞后多个小版本，预计将来同步。 |
| 节日活动 | 标准 | 同步 |
| 文本语言 | 中文 | 国际服：日、英、德、法；韩服：韩文；繁中服：繁体中文 |
| 配音语言 | 中、日 | 国际服：日、英、德、法；韩服：韩语；繁中服：日语 |
| 区服信息 | 分4个大区，共27个服务器 | 国际服：按北美、欧洲、日本、大洋洲分4个DC，共10个大区，81个服务器 |
| 玩家群体 | 以国内玩家为主，有部分留学生，及其他华语地区玩家。 | 各DC以地域各自分布地区，但实际各地人群都有，也有部分非运营地区玩家（中韩新马等）。 |
| 购买方式 | 点卡/月卡，无需 CDKey | CDKey + 月卡，没有点卡 |
| 额外雇员 | 买断 | 月租 |
| 运行平台 | 仅 Windows(盛趣，WeGame) | Windows(Steam), MacOS, PS4, PS5 （账号数据互通，游戏需每平台单独购买） |
