# 阿修罗歼灭战

前面出啥躲啥即可。

读条==化身开眼==之后，BOSS会出现一个巨大分身到场边。

BOSS和场边大BOSS连线时，BOSS使用技能，被连线的大BOSS会读==化身剑技==并复读刚才小BOSS用的技能，且范围更大。

BOSS读条==三头六臂==，会出现6次半场刀预兆，先站前两刀预兆的安全区，第2、4、5刀后左右移动即可。

读条==愤怒的神气/喜悦的神气==时，场地分成红蓝两半，同时画面中央会闪现一个红色或蓝色面具，前往与面具反色的地板躲避。如果没来得及看清面具，愤怒踩蓝，喜悦踩红。

躲过几波AOE后，BOSS会再次与场边分身连线并使用技能，这次分身会先使用==化身转移==移动到其他位置，再复读技能。
