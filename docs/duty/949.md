# 高贝扎歼灭战

BOSS读条==迟缓地爆==后，身上会出现一个陨石，缓缓移动到场地一角，远离那个角落即可。

读条==弦月剑==时，会随机面向一名玩家，并使用正面180° AOE，该技能会多次使用。

读条==极寒突袭==时，场地上会出现两面冰墙，读条结束后冰刺发射，各自覆盖1/4半场，躲开即可。

读条==虚空陨石==，点名2<Role name="tank" />T范围死刑，判定时会陆续落下5颗陨石，其中第5颗伤害最大，注意保持躲开其他玩家。

读条==迟缓电火花==会点名随机若干名玩家放出3连AOE黄圈，出啥躲啥即可，但注意接下来会立刻接弦月剑（半场刀），不要跑丢。

读条==咒缚寒气==后会释放全屏AOE+dot，<Role name="healer" />治疗注意抬血。

BOSS使用==黑龙剑阿珠达雅==后，转场动画进入P2，放全场AOE，<Role name="healer" />治疗注意抬血。

读条==黑龙弦月剑==后，BOSS会在半场弦月剑之后紧接脚下范围AOE，AOE范围大概会覆盖地面的圆形花纹，比近战最远距离更远。

之后会有一系列复杂机制组合，按如下处理：
* 被点名击退的<Role name="healer" />治疗站在场地中央，击退后向北（BOSS方向）走两步，等出现直线AOE预兆后向南贴边。
* <Role name="tank" /><Role name="healer" /><Role name="dps" />其他人靠近被点名的治疗，大体均分左右击退踩塔，塔有1/3塔或者2/2人塔，只要塔里人多于塔中柱子数量即可。踩完塔后去南场躲避。
如果塔中人数少于塔的数量，会造成巨量全屏AOE伤害，需要<Role name="healer" />治疗及时抬血，否则会被后续陨石团灭。

之后BOSS会向东西南北依次放出4个分身，需要记住分身出现的顺序。然后场地四方出现风球，风球会按照分身出现的顺序施放直线AOE，需要按顺序插缝躲避。

读条==虚空彗星雨==后，场地上会依次出现步进式AOE预兆，在最后一个预兆等待第一个AOE位置陨石落下后，移动到第一个AOE位置。

在黑龙加强版迟缓电火花之后，BOSS会读条==集束黑龙闪==是点名两名治疗的直线分摊AOE，<Role name="healer" />治疗左右分开，<Role name="tank" /><Role name="dps" />其他人分别去参与分摊即可。

之后全都是机制复读，唯独要注意黑龙弦月剑之后跟的AOE范围很大，但同时有分散或分摊。
