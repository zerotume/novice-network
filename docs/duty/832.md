# 艳翼蛇鸟歼灭战

开场后不久需要根据蛇鸟动作和屏幕提示躲避AOE：
* 蛇鸟起飞绕着场地飞行：场地中间安全
* 蛇鸟发红向上站起，两个爪子向前伸：远离蛇鸟目标圈，该AOE范围超过<Role name="melee" />近战最远输出距离。
* 蛇鸟发红并向后退，两边翅膀展开，张开大嘴：范围极大的扇形AOE，需要往场地左上角或右上角躲避，安全区较小。

之后蛇鸟读条改变场地，有2种情况：
## 冰场地

切换为冰雪场地后不久，屏幕上会出现提示**东南方传来雪崩逼近的巨响……**，同时右后方传来轰隆隆的声音。观察右后方的雪山，会有蓝色箭头指向东北半场或西南半场，观察箭头并躲开即可。

读条==结冰尘==后，所有人需要保持移动2s，否则就会被冻住。

之后蛇鸟会在开场3种机制（月环、远离、扇形）中随机使用一个，然后场地出现2个魔法阵，并使用米字型8方直线攻击，找缝隙躲避即可。

## 雷场地
雷场地为4x3的格子，其中6块发光，踩上去会浮空 

先站在发光格子上躲全屏AOE，然后场地落下4根羽毛，离开发光格子和有羽毛的格子躲雷云（若在浮空状态，头顶会出现雷云）。

最后场地左侧或右侧出现一排魔法阵，找到缺口躲避即可。

## 场地复原

第一次场地复原后，会出现双人塔+击退+场边AOE预兆。<Role name="tank" />双T先开减伤踩塔，同时所有人开防击退即可。若没有防击退技能，需要前往场地最前端，并注意不要被击退到后场的AOE里。

第二次场地复原后，场地会出现3个塔，同时附带场地扇形AOE和分摊直线，所有人在一起移动分摊，并把塔都打掉。之后读条赋予所有玩家<Status :id="4122" name="灾祸之炎" />，并从头开始重放机制。<Role name="healer" />治疗注意从这里开始机制会变得比之前更疼，需要更多减伤。

机制重放时：
* 冰场地会同时出现雪崩+冰魔法阵，需要在安全半场找安全区。
* 雷场地会同时出现全屏AOE+魔法阵，需要在安全列找发光地板躲避。
